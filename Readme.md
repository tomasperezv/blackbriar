Blackbriar
==========
Node.js MVC + API engine

### Run the tests ###

Unit tests are implemented in [nodeunit](https://github.com/caolan/nodeunit)

```shell
sudo npm install nodeunit
```

```shell
$ nodeunit test/server/
router-test
✔ isApiRequest
✔ isReverseProxyRequest
...
OK: 29 assertions (401ms)
```
